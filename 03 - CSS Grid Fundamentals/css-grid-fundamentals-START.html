Notes: In this tutorial we learn that to apply grid to an element we set the parent to 'display: grid;' and then we can set the amount of columns with 'grid-template-columns: 200px 200px' and the number of measurements we tag on it will create a column for each. This one will make 2 - 200px columns.

Then we learned we can create spacing between these elements by using 'grid-gap: 30px;'. This makes a margin around ALL the elements of 30px.

Then we can use 'grid-template-rows: 400px;' to set the height of each row. This one will set the first row to 400px.

we also learned a css shorthand for applying a large number of rules to the same row or column. grid-template-rows: repeat(5, 100px); This says to make 5 rows each at 100px.
